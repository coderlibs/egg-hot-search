# egg-hot-search
该项目由[`coderlibs `](http://coderlibs.com)官方出品
基于egg开发的后端热搜爬虫服务，支持微博，百度，头条，知乎等热搜爬虫

## 环境
Node.js 16.0+

## 安装依赖
```bash
    cnpm install  /  npm install
```

## 运行
```bash
    npm run dev   // mac本地开发调试
    npm run win:dev   // win本地开发调试
    npm start // 启动线上服务
    npm stop  // 停止线上服务
```

运行后，访问http://127.0.0.1:8848
